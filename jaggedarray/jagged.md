 A jagged array is an array of arrays where each inner array can have a

different length. Unlike regular multi-dimensional arrays (like 2d arrays),

where each row has the same number of columns, jagged arrays allow rows of

varying sizes.

 They have a flexible memory usage since each row can be sized

independently.

 Useful when data naturally has varying lengths per row.

 Note that : a jagged array is different from Regular multidimensional array

where; Regular Multidimensional array has fixed number of columns for each

row, or fixed number of elements of each dimension.
